# World Cup 2026 → TikTok Daily Video Bot

Auto-generates one original, fully-licensed vertical video per day for the 2026 World Cup: today's fixture, head-to-head country stats, three AI-written facts, and an engagement CTA — then posts it to TikTok.

No reposted YouTube content. Every frame is generated programmatically, so there's nothing for TikTok's duplicate-content detection to flag.

## How it works

```
GitHub Actions (daily cron)
   │
   ├─ scripts/fetch-data.mjs
   │     API-Football  → today's fixture (teams, kickoff, venue)
   │     REST Countries → capital, population, flag
   │     Claude API     → hook, 3 facts, prediction question
   │     ↓ writes data/today.json
   │
   ├─ remotion render → out/today.mp4 (1080×1920, 28s)
   │
   └─ scripts/post-tiktok.mjs → TikTok Content Posting API
```

## Quick start (local)

```bash
npm install
npm run preview     # opens Remotion Studio — live-edit the template
npm run render      # renders out/today.mp4 using data/today.json
```

The repo ships with sample data (Brazil vs Germany) so it renders with zero API keys.

To run the full pipeline:

```bash
cp .env.example .env   # fill in your keys
npm run daily          # fetch → render → post
```

## Deploy the daily automation (video → your phone at 7am)

GitHub here is just the free always-on computer that does the 7am work — you set it up once and never look at it again. The video arrives on your phone via Telegram every morning.

**One-time setup (~20 minutes):**

1. **Create a Telegram bot** (your delivery channel): message @BotFather in Telegram → `/newbot` → copy the token. Send your new bot any message, then open `https://api.telegram.org/bot<TOKEN>/getUpdates` in a browser and copy your chat `id`.
2. **Get a free API-Football key** at api-football.com (fixtures) and an Anthropic API key at console.anthropic.com (fact writing).
3. **Upload this folder to a new private GitHub repo** (github.com → New repository → "uploading an existing file" drag-and-drop works, no git knowledge needed).
4. **Add the secrets** in the repo: Settings → Secrets and variables → Actions → New repository secret. Add: `API_FOOTBALL_KEY`, `ANTHROPIC_API_KEY`, `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`.
5. **Test it immediately:** Actions tab → "Daily World Cup video → your phone" → Run workflow. Within ~5 minutes the video should appear in your Telegram.

From then on it runs automatically at 7:00 AM (WAT) daily. Each morning: open Telegram → save the video → post to TikTok with the suggested caption included in the message.

**Optional later:** add the three `TIKTOK_*` secrets to enable fully automatic posting once your TikTok developer app is approved. Until then that step skips itself silently. Note posts go up as private (`SELF_ONLY`) until the app passes TikTok's audit — then switch `privacy_level` to `PUBLIC_TO_EVERYONE` in `scripts/post-tiktok.mjs`.

## TikTok API approval (the slow part)

- Register at developers.tiktok.com → create an app → apply for **Content Posting API** with the `video.publish` scope. Manual review, typically 1–2 weeks.
- **Faster interim option:** skip the TikTok step and push `out/today.mp4` to a scheduler (Metricool, Postiz, Buffer) via their API, or simply download the artifact from the Actions run and post manually — the generation is still fully automated.

## Customizing the video

Everything visual lives in `src/MatchFactsVideo.tsx`:

- **Scenes:** Hook (3.5s) → Diagonal VS (6.5s) → 3 fact cards (4.5s each) → Outro CTA (4.5s)
- **Design tokens:** `PITCH`, `CHALK`, `GOLD` constants at the top
- **Fonts:** Anton (display) + Inter (body) via `@remotion/google-fonts`
- Fact-card borders automatically pick up each team's color from the data

Add voiceover by generating TTS audio in `fetch-data.mjs` (ElevenLabs/OpenAI) and dropping an `<Audio>` tag into the composition. Add royalty-free music the same way.

## No-fixture days

`fetch-data.mjs` exits gracefully and keeps the previous `today.json`. To avoid reposting, either let the job fail (`exit 1`) on those days or maintain a small queue of evergreen fact videos (past finals, host-city facts, team profiles) and rotate through it.

## Compliance notes

- Flags from flagcdn.com are public domain.
- Don't add FIFA logos, official broadcast clips, or match footage — FIFA enforces aggressively.
- Facts generated by Claude should be spot-checked occasionally; LLMs can get sports trivia wrong, and wrong facts in comments hurt more than no post.
